关于GUET Online Judge

        本系统旨在为热爱程序设计与算法分析的爱好者提供一个练习和交流的平台，24小时开放。您可以使用你喜欢的语言解题，如C/C++，Java，Ruby，Python等。GUET OJ同时支持Virtual Judge，我们目前已经支持HDOJ虚拟判题，后续将添加更多OJ平台的虚拟判题。

        用户可以在题库（Problemset）中选择题目练习，并可以在实时提交状态（Status）中看到所有提交，可以在竞赛（Contest）中参加比赛，话题（Topic）则为用户提供话题交流的平台，更多功能还有待用户您去发现，希望各位在此玩得开心。
        OJ系统运行于桂林电子科技大学，数学与计算科学学院实验室，并由acmlgt  &  ACSolo  &  Jungle 创建并管理，若您有任何意见和想法可发站内信给@ACSolo。

        如果您想在OJ举办比赛，请发邮件 Zengke Wei，题目和测试用例请自备。

OJ是开源项目
        OJ为开源项目，您可以自由下载全部源码并安装部署（详情见Topic），但请保留原作者信息。

        如何获取：

  Use this command to anonymously check out the latest project source code:
# Non-members may check out a read-only working copy anonymously over HTTP.
svn checkout http://acsolo-oj.googlecode.com/svn/trunk/ acsolo-oj-read-only

OJ使用建议

        我们强烈建议您使用 Chrome 12+、FireFox 8+、Safari 4.0+、Opera 9.5+、 IE 8+浏览器；我们将不会修复低于IE 8出现的兼容问题，因为我们讨厌这些浏览器。
